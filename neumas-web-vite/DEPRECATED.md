# ⚠️ DEPRECATED

This directory (`neumas-web-vite/`) contains the original Vite + React 18 frontend for Neumas.

**This frontend is DEPRECATED and receives no new development work.**

## What to use instead

All frontend development happens in `neumas-web/` (Next.js 16 / React 19).

## Why deprecated

- The Next.js frontend (`neumas-web/`) has feature parity and is actively developed
- The Vite frontend has diverged and is not maintained
- All operator features, auth improvements, and UI redesigns are in `neumas-web/` only

## Removal

This directory will be removed in a future cleanup sprint. Do not reference it in new code, documentation, or CI pipelines.

See `ARCHITECTURE.md §13` for details.
